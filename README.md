## プロジェクト概要

印西市の子どもたちと共に育てた酒米「総の舞」を使用した日本酒のプロモーションサイトです。

### 主な機能

- レスポンシブデザイン（PC/タブレット/スマホ対応）
- 龍の動画アニメーション
- microCMSと連携したお知らせ機能
- 生産者セクション（左右分割レイアウト）

## ページ構成

1. **ヘッダー** - ロゴ・タグライン
2. **ヒーロー** - 龍アニメーション + ロゴ文字
3. **お知らせバナー** - 最新のお知らせを表示
4. **商品紹介** - 選ばれし米、磨かれし技
5. **プロジェクト** - 地域と未来をつなぐプロジェクト（田植え・稲刈り・醸造）
6. **味わいの特徴** - 3つの特徴カード
7. **商品情報** - 商品詳細・スペック表
8. **生産地の恵み** - 米の産地・龍伝説
9. **生産者** - 窪田酒造・米のはすみ（ロゴ・写真付き）
10. **取扱店舗一覧** - 店舗テーブル
11. **お知らせ** - microCMS連携
12. **お問い合わせ** - 連絡先
13. **フッター**
